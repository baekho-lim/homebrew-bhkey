# homebrew-bhkey

Homebrew tap for [bhkey](https://github.com/baekho-lim/bhkey) — zero-latency modifier key remapper for macOS external keyboards.

## Usage

```bash
brew tap baekho-lim/bhkey
brew install bhkey
```

## What is bhkey?

bhkey remaps modifier keys on external keyboards using Apple's native `hidutil` kernel API — no virtual driver, zero input latency.

See [baekho-lim/bhkey](https://github.com/baekho-lim/bhkey) for full documentation.
